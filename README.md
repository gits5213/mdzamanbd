# Md. Shahnewaz Zaman - Political Campaign Website

A modern, responsive political website built with Next.js for Md. Shahnewaz Zaman (Shanto), candidate for Sherpur-2 constituency (Nakla-Nalitabari), Bangladesh.

## Features

- Modern, clean design
- Responsive layout for all devices
- Multi-language support (Bangla/English)
- Political vision and objectives
- Social activities showcase
- Constituency information (Nakla & Nalitabari)
- Contact information
- News and updates section

## Technology Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- React Icons

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view the website.
